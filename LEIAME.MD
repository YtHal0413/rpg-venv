# Plataforma RPG - Criador de Fichas

## Descrição do Projeto

O projeto consiste em uma plataforma universal voltada para a criação e o gerenciamento de fichas de personagens de RPG em cenários de grupo, visando suprir a falta de plataformas dedicadas a sistemas de nicho ou próprios.

Tal proposta foi estruturada para integrar, em um único projeto, os conteúdos das principais disciplinas do curso de Tecnologia em Sistemas para Internet: Programação Orientada a Objetos, Programação para Web e Banco de Dados.

## Arquitetura e Tecnologias

- Backend: Python orientado a objetos.
- Framework web: Flask.
- Estilização: Tailwind css
- Banco de dados: SQLite.
- Frontend: JavaScript, HTML e CSS.
- Estrutura de dados: colunas, abas, seções e componentes de ficha.
- Templates armazenados em JSON para reaproveitamento de estruturas.

## Objetivos

- Disponibilizar uma ferramenta de criação e edição de fichas de RPG flexível e customizável.
- Permitir a construção de fichas através de colunas, abas e seções especializadas.
- Oferecer templates salvos para replicação rápida de estruturas de ficha.
- Integrar criação de campanhas, biblioteca pessoal de documentos e sistema de gerenciamento de dados.
- Suportar sistemas de RPG de nicho ou regras próprias que não têm ferramentas prontas.

## Principais Funcionalidades

- Criador customizável de fichas com estrutura em colunas, abas e seções.
- Sistema de templates para salvar e aplicar layouts de fichas.
- Interface dinâmica em JavaScript para edição rápida e visualização imediata.
- Controle de rotas e APIs em Flask para a lógica de backend.
- Persistência de dados usando banco SQL com SQLite.
- Sistema de bandeja de dados para registrar valores, contadores e checkbox em fichas.
- Upload e anexo de documentos pessoais na biblioteca do usuário.
- Criação e edição de campanhas de RPG com controle de jogadores e personagens.
- Autenticação básica de usuários com perfis e avatares.

## Como usar

1. Faça o Download do Repositório
2. Ative o ambiente virtual do projeto.
3. Instale as dependências com `pip install -r requirements.txt`.
4. Execute a aplicação Flask via terminal.
5. Acesse o sistema no navegador para criar e gerenciar fichas de RPG.
